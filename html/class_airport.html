◆ Airport()

Airport::Airport ( const vector< string > &  dataRead)

Crea un aeroport a partir d'un vector de strings.

Precondició
La mida del vector dataRead és 14. Les dades estan en el següent ordre: 
        ID, nom, ciutat, pais, IATA, ICAO, latitud, longitud, altitud,
        TimeZone, DST, databaseTZ, tipus, fontDades.
Postcondició
Crea un nou aeroport a partir dels valors continguts al vector dataRead. Per a tot valor desconegut se li assigna un si es tracta d'una cadena de caràcters o un -1000 si és un valor numèric.
