◆ addAirport()

void Airports::addAirport ( const vector< string > &  dataRead)

Afageix un aeroport al conjunt d'aeroports.

Precondició
La mida del vector dataRead és 14. Les dades estan en el següent format: 
             ID, nom, ciutat, pais, IATA, ICAO, latitud, longitud, altitud,
             TimeZone, DST, databaseTZ, tipus, fontDades.
El codi ID, el codi IATA i el codi ICAO han de ser identificatius per cada aeroport, és a dir, no pot haver-hi dos aeroports amb algun dels camps esmentats iguals entre els dos. No obstant, si és possible no conèixer el codi IATA o ICAO d'un aeroport, però sempre cal proporcinar el ID.
Postcondició
Crea un aeroport a partir dels valors del vector dataRead i l'afageix al conjunt d'aeroports. El codi ID és identificatiu per a cada aeroport. També ho és el codi IATA i ICAO si es disposa d'ells.

Comprova a partir del ID, el codi IATA o el codi ICAO si existeix l'aeroport dins el conjunt.

Precondició
ID és -1 si es desconeix. Si es disposa del dos parametres, aquests han de fer referència al mateix aeroport.
Postcondició
Retorna cert si dins del conjunt hi ha una aeroport que tingui com a ID el valor d'entrada ID. Si aquest és desconegut, s'intenta identificar per codi IATA o ICAO. Es retorna fals quan per cap dels paràmetres s'ha pogut comprovar l'existència de l'aeroport dins el conjunt.

Precondició

Retorna cert si dos aeroports, un de sortida i l'altre d'arribada, estan ubicats al mateix país d'una aerolínia, fals altrament.

Precondició
IDAirportSource i IDAirportDeparture són -1 si es desconeixen; pel contrari, IATAICAO sempre seran valors proporcionats, ja que s'usen com a recolzament per a identificar els aeroports en cas de no disposar de ID. Els aeroports han d'existir dins el conjunt d'aeroports. countryAirline és el país amb el que es compara; país de l'aerolinia
Postcondició
Retorna cert si els dos aeroports, identificats pel codi ID com a primera opció o pel codi IATAICAO com a segona, es troben al mateix país que l'aerolínia, fals altrament.
