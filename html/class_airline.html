Airline::Airline ( const vector< string > &  dataRead)

Crea una aerolínia a partir d'un vector de strings.

Precondició
La mida del vector dataRead és 8. Les dades estan en el següent ordre: 
     ID, nom, alies, codiIATA, codiICAO, callsing, país, activa.
Postcondició
Crea una nova aerolínia a partir dels valors continguts al vector dataRead. Per a tot valor desconegut se li assigna un \N.
