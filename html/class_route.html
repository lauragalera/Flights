Route::Route ( const vector< string > &  dataRead)

Crea una ruta a partir d'un vector de strings.

Precondició
La mida del vector dataRead és 9. Les dades venen ordenades de la forma següent: 
               IATAAerolinia, codiIDAerolinia, IATAAeroportSortida,
               codiIDAeroportSortida, IATAAeroportArribada, codiIDAeroportArribada,
               codeShare, parades, equipament.
*
Postcondició
Crea un nova ruta a partir dels valors continguts al vector dataRead. El codi ID aerolinia i els codis ID aeroport són -1 si el valor és desconegut.
