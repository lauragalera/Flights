Afageix un vol al conjunt de vols sempre i quan aquest sigui vàlid.

Precondició
La mida del vector dataRead és 20. Les dades estan en el següent ordre: 
            any, mes, dia, diaMes, IATAAerolínia, Numvol, IATAAeroportSortida, ciutatAeroportSortida,
            IATAAeroportArribada, ciutatAeroportArribada, horaEsperadaSortida,horaSortida, retardSortida,
            horaEsperadaArribada, horaArribada, retardArribada, cancelat, duradaEsperadaVol, duradaVol
            distancia.
Postcondició
Donat un vector de strings amb els valors d'un vol, es crea el vol i s'afageix al conjunt de vols després de verifica si aquell vol és valid, comprovant si existeix la companyia aeria que l'opera i tant l'aeroport de sortida com el d'arribada.

Retorna un vector de n parelles, ordenat decreixentment pel primer, amb el primer argument la suma de retards d'arribada o sortida dels vols i com a segon una tupla InfoAirline fent referència a l'aerolínia.

Precondició
El vector dataRead conté quantes aerolínies llistar (0 per totes). type és 'D' per indicar retards de sortida o 'A' per indicar retards d'arribada.
Postcondició
Retorna un vector de n parelles on el primer valor representa la suma de retards de sortida o bé d'arribada -depent del valor de type- de tots els vols que té una aerolínia. L'aerolínia ve identificada a partir de la tupla InfoAirline on hi consta el nom i el país de l'aerolínia, ocupant el segon lloc de la parella. El vector està ordenat decreixentment pel primer valor de la parella i es retornen totes o les n primeres aerolínies amb retards.

Retorna un vector de n parelles, ordenat decreixentment pel primer, amb el primer argument la suma de retards d'arribada o de sortida dels vols i com a segon una tupla InfoAirport fent referència a l'aeroport.

Precondició
El vector dataRead conté quants aeroports llistar (0 per tots), any, mes, dia del mes. type pot ser 'D', per indicar retards de sortida, o 'A' per indicar retards d'arribada. Si la data no existeix és llença una excepció.
Postcondició
Retorna un vector de n parelles on el primer valor representa la suma de retards de sortida o bé d'arribada -depenent del valor de type- de tots els vols que té un aeroport en la data indicada a dataRead. L'aeroport ve identificat a partir de la tupla InfoAirport amb el nom, la ciutat i el país, ocupant el segon lloc de la parella. El vector està ordenat de forma decreixent pel primer valor de la parella i es retornen tots o els n primers aeroports amb més retards.
